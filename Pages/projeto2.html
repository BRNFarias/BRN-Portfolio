Visão Geral

Este projeto é uma ferramenta de RPA (Robotic Process Automation) desenvolvida para escritórios de contabilidade e BPO Financeiro. O objetivo principal é automatizar o processo repetitivo de download das guias de pagamento (DAS) do Simples Nacional para múltiplos clientes MEI.

O grande diferencial técnico foi a superação das defesas anti-robô do portal do Governo. Utilizando a biblioteca moderna DrissionPage, a automação controla o navegador via protocolo CDP, tornando a navegação indetectável e muito mais fluida que o Selenium tradicional.

Destaques Técnicos

  • Controle de navegador via CDP (Chrome DevTools Protocol) para evitar detecção.
  • Leitura e processamento em lote a partir de planilhas Excel (Pandas).
  • Sistema inteligente de renomeação de arquivos PDF baseada no CNPJ.
  • Lógica híbrida (semi-automática) para resolução de Captchas quando necessário.
